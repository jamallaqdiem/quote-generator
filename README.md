A full-stack application that generates random inspirational quotes.
Users can view random quotes from a pre-defined list and add their own new quotes via the backend.

Live DemoFrontend:
frontenedapp.hosting.codeyourfuture.io
Backend: firstdeployedapp.hosting.codeyourfuture.io

Tech Stack
Frontend: React (Vite), JavaScript
Backend: Node.js, Express, CORS
Deployment: Coolify

Project Structuretext/
├── frontend/ # React client code
└── backend/ # Express server code

Setup and InstallationPrerequisites
Node.js (v22.12.0 or higher)
npm
Local Development

Clone the repository:
git clone

Bash:
cd quote-generator

Run Backend:
cd ../backend
npm install
node main.js

Run Frontend:
cd ../frontend
npm install
npm run div
